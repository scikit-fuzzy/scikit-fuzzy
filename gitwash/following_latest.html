Following the latest source¶

These are the instructions if you just want to follow the latest scikit-fuzzy source, but you don’t need to do any development for now.

The steps are:

  • Install git

  • get local copy of the scikit-fuzzy github git repository

  • update local copy from time to time

Get the local copy of the code¶

From the command line:

git clone git://github.com/scikit-fuzzy/scikit-fuzzy.git

You now have a copy of the code tree in the new scikit-fuzzy directory.

Updating the code¶

From time to time you may want to pull down the latest code. Do this with:

cd scikit-fuzzy
git pull

The tree in scikit-fuzzy will now have the latest changes from the initial repository.
